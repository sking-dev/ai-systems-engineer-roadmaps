## Getting Started

Begin with [`cloud_to_ai_systems_framework.md`](docs/cloud_to_ai_systems_framework.md) for a high-level overview of the transition journey.

Before diving into the roadmaps, review [the Foundations section](#foundations) to see how existing cloud platform skills map onto AI systems engineering.

Then explore the year-by-year roadmaps to develop a more detailed understanding of the skills and capabilities needed at each stage.

As this roadmap evolves, additional documents will be added to provide more detailed guidance for each stage of the journey.
